## Secret Hitler Mobile

A mobile-friendly, browser-based adaptation of Secret Hitler, designed for in-person play using a single shared device.

This project automates role assignment, policy decks, government formation, special powers, victory conditions, and expansion content while preserving the social deduction experience of the original board game.

The entire application runs locally in the browser as a single standalone HTML file. No installation, server, or internet connection is required after downloading.

# Features

- Complete implementation of Secret Hitler
- Mobile-first interface optimized for phones and tablets
- Single-device gameplay for in-person groups
- Automatic role assignment and game setup
- Policy deck and discard pile management
- Election and legislative phases
- Fascist powers and victory conditions
- Multi-language support
   (Just english and Brazilian portuguese for now. You may contribute by adding more languages)
- Readable font mode
- Support for the Socialist Expansion
- Optional device-passing mode for Bugging and Recruitment powers
- Fully self-contained HTML application

# Running the Project

# Option 1: Run Locally

Download the HTML file and open it in any modern web browser.

No installation, dependencies, or server setup is required.

# Option 2: GitHub Pages

This project is hosted directly in GitHub Pages.

Play here: https://brluquinha19.github.io/Secret-Hitler-In-Person-Play/

# How to Play

1. Open the html file or the link.
2. Configure the desired game settings.
3. Enter the player names.
4. Start the game.
5. Follow the instructions displayed on screen.

The application manages the game flow while players perform discussion, voting, deduction, and social interaction in person.
